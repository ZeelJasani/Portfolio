Learnix is a modern, full-stack Learning Management System (LMS) built with cutting-edge web technologies. It provides a comprehensive platform for creating, managing, and delivering online courses with advanced features like drag-and-drop course structuring, rich text editing, and secure file management.

## Tech Stack

### Frontend
- **Framework:** Next.js 15.4.6 (App Router)
- **Language:** TypeScript 5.0
- **Styling:** Tailwind CSS 4.0 + CSS Modules
- **UI Components:** Radix UI + shadcn/ui
- **State Management:** React Query + Zustand
- **Forms:** React Hook Form + Zod
- **Icons:** Lucide React + Tabler Icons

### Backend
- **Runtime:** Node.js 18+
- **API:** Next.js API Routes
- **Authentication:** NextAuth.js (Better Auth)
- **File Storage:** AWS SDK v3
- **Email Service:** Resend
- **Security:** Arcjet

### Database
- **Primary Database:** PostgreSQL 16
- **ORM:** Prisma (Type-safe database client)
- **Migrations:** Version-controlled with Prisma Migrate

### DevOps & Tools
- **Hosting:** Vercel
- **CI/CD:** GitHub Actions
- **Containerization:** Docker
- **Linting/Formatting:** ESLint + Prettier
- **Testing:** Jest + Playwright + React Testing Library

## Key Features

- **User Authentication:** Secure login with multiple providers (GitHub, email OTP)
- **Course Management:** Create and organize courses with chapters and lessons
- **Rich Content Editor:** Advanced text editing with TipTap integration
- **File Management:** Secure file uploads with S3-compatible storage
- **Responsive Design:** Mobile-first approach with modern UI components
- **Admin Dashboard:** Comprehensive course administration tools
- **Payment Processing:** Integrated Stripe payment system

## Usage

### For Learners
- **Browse Courses:** Visit the public landing page to explore available courses
- **User Registration:** Sign up using email or GitHub OAuth
- **Course Access:** Enroll in courses and track progress
- **Learning Interface:** Access course content through organized chapters and lessons

### For Instructors/Admins
- **Admin Access:** Login with admin credentials
- **Course Creation:** Use the admin dashboard to create new courses
- **Content Management:** Organize courses into chapters and lessons
- **Media Upload:** Upload images and videos using the integrated file manager
- **Course Publishing:** Manage course status (draft, published, archived)

## Key Functionalities
