## üöÄ Getting Started

1. **Clone the repo:**

   ```bash
   git clone https://github.com/ZeelJasani/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**

   ```bash
   pnpm install
   ```

3. **Run development server:**

